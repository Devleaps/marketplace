# DevLeaps Marketplace

Plugin marketplace for Claude Code.

## Setup

Add the marketplace:

```bash
claude plugin marketplace add Devleaps/marketplace
```

Install a plugin:

```bash
claude plugin install agent-policies@Devleaps-marketplace
```

## Plugins

| Plugin | Description |
|---|---|
| `agent-policies` | Forwards hook events to a policy server that can allow, deny, ask, or halt agent actions |
